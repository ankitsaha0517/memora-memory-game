# 🧠 Memora

> Train Your Memory. Sharpen Your Mind.

A premium Simon-inspired memory game built using **HTML**, **CSS**, and **Vanilla JavaScript**. Memora challenges players to remember increasingly complex color sequences while providing a polished user experience with modern UI design, smooth animations, and responsive gameplay.

---

## ✨ Features

- 🎮 Classic Simon-style memory gameplay
- 📈 Progressive difficulty system
- 🏆 High score tracking
- 💾 LocalStorage support
- ✨ Modern Glassmorphism UI
- 📱 Fully responsive design
- ⚡ Smooth animations
- 🎵 Interactive sound effects *(optional)*
- 🌙 Dark mode interface
- 📊 Real-time score and level tracking

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
